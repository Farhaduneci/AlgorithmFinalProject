Project Description

Given the fixed array of numbers, representing some dots, from 1 to 5. and a random set of numbers in the same range, you must find the MAX subset that if we want to draw a line between the dots no two line overlap .

Solution

If we find a increasing sequence of numbers is the input array, They will never cause overlap, the first array is always sorted, so any increasing subset will always be the best answer.

Input

Enter the random sequence, Separate by Space
